# Worker workshop

1. `git clone https://github.com/quadristan/nodejs-paris-workshop-sworker.git`
2. `npm install --ignore-scripts`

# In case you need HTTPs

3. Install [mkcert](https://github.com/FiloSottile/mkcert)
   1. macos `brew install mkcert`
   2. linux btw `sudo pacman -Syu mkcert`
4. `mkcert -install && mkcert localhost 127.0.0.1 ::1`
